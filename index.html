USER GUIDE
Platform overview, workflow, and role-based feature map
◈
1. Screen a Deal
Go to Screen in the top nav. Select the asset class (Debt, PE, CIS, REIT), enter financial inputs, and click Run Screen. Results appear instantly with PASS/WATCH/FLAG status on every metric, a composite score ring, and decision rule.
⚙
2. Doc Intelligence
Go to Doc Intelligence. Paste any financial statement, annual report, or fund fact sheet into the text area. Click Extract Metrics — the engine parses key financial figures and auto-fills the screening inputs. Then click Run Screen.
◎
3. Macro Dashboard
The Macro view shows live Ghana rates — BoG policy rate, T-bill yields, lending rates, CPI, GDP. Click Update Rates to enter the latest BoG data. Thresholds in all screens update dynamically with the macro context.
▣
4. Deal Library
All saved screens appear in the Library. Filter by segment, verdict, or search by name/analyst. Click any row to open the full deal detail modal. VPs and above see all team deals; Analysts see their own.
✦
5. IC Memo
Available to Associates and above. Select any saved deal and click Generate IC Memo. The platform formats a full Investment Committee memorandum with metric table, risk flags, macro context, and recommendation.
⊞
6. Admin Panel
Available to Directors and MDs. Manage users, assign roles, view firm-wide analytics, and access the full permissions matrix. Reset data or promote/demote users from this view.

Screening Decision Rules
Proceed to Full DD
0–1 FLAG, ≤2 WATCH across all quantitative metrics. Clean screen — proceed to legal, operational, and financial due diligence.
Proceed With Conditions
1 FLAG or 3+ WATCH. Address root cause. Require structural mitigants or enhanced monitoring covenants.
Decline
2+ FLAGS. Decline or return to originator for material restructuring. Document rationale for audit trail.
